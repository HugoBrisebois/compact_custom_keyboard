## Compact Custom Keyboard

This is a compact Keyboard the is powered by a raspberry pi pico and is a hotswapable kailth choc mini keyswitches, the keyboard also has LEDs that can be used for a aestetic lighting and as light to see what you are typing.
The Keyboard is also a 60% keyboard meaning that it is quite small while not having too many layers like a 40% one.

# Features
- powered by a raspberry pi pico
- RGB LEDs
- kailth choc mini keyswitches
- 3d Printed keyswitches

# Reason for build this project
i wanted a keyboard that is completely custom to me and is also something that i can use in a cyberdeck project that i want to build later on. this keyboard is also perfect for everyday use and i am looking forward to using this keyboard too.
